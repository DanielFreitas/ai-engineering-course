# AI Engineering Course

Bem-vindo ao **AI Engineering Course** — um currículo estruturado para engenheiros que querem construir **produtos reais com LLMs**, do fundamento conceitual até a operação em produção.

## O que você vai aprender

Este curso está organizado em **três trilhas**:

| Trilha | Módulos | Foco |
|--------|---------|------|
| **Núcleo** | 1–7 | Fundamentos de AI Engineering, APIs, prompts, RAG, avaliação, segurança e arquitetura |
| **Backend** | 8–16 | FastAPI, contexto/memória, tool calling, persistência, async, cache, observabilidade, LLMOps e segurança operacional |
| **Frontend** | 17–20 | React com IA, UX, integração frontend-backend e telemetria |

## Pré-requisitos

- Experiência com desenvolvimento backend ou frontend (qualquer linguagem)
- Conhecimento básico de APIs REST
- Python intermediário é útil, mas não obrigatório para todos os módulos

## Como usar este material

1. Comece pela trilha **Núcleo** — ela é pré-requisito para as demais
2. Após o checkpoint do núcleo, você pode seguir **Backend** e **Frontend** em paralelo
3. Cada módulo tem uma **prática independente** — simples, curta e fácil de validar

## Grade completa

### Núcleo (obrigatório)
1. Fundamentos de aplicações com IA
2. Integração com modelos e APIs de IA
3. Engenharia de prompt e saídas confiáveis
4. Embeddings, recuperação de informação e RAG
5. Avaliação de sistemas com LLMs
6. Segurança, confiabilidade e governança
7. Visão geral da arquitetura com IA

### Backend
8. FastAPI para features com IA
9. Contexto, memória e estado
10. Tool calling e orquestração
11. Persistência e decisões arquiteturais
12. Arquiteturas assíncronas com Pub/Sub
13. Cache e otimização com Redis
14. Observabilidade com OpenTelemetry e Datadog
15. LLMOps e ciclo de vida operacional
16. Segurança operacional para aplicações com IA

### Frontend
17. Frontend de aplicações com IA em React
18. UX e qualidade percebida
19. Integração frontend-backend para fluxos com IA
20. Telemetria orientada à experiência
